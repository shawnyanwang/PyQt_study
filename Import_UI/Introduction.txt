After the "*.ui " file generated by Qt designer, we need to import the UI file into python main program to achieve our design. This folder include the simple examples to show how to import UI.  
当用qt designer 生成UI文件以后，我们需要把UI文件导入到python主程序中，使主程序可以直接加载设计好的UI。该文件夹包括了相应的范例。


Configuration (配置): 
python == 3.7.3
PyQt5 == 4.19.15
pyqt5designer == 4.19.5
